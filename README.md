# SpringBoot Project

### 📢 프로젝트 설명
- 개인프로젝트<br>
    - SpringBoot를 활용한 쇼핑몰 프로젝트 입니다.
- 구현 목표<br>
    - 사용자 : [조회, 장바구니, 결제]
    - 판매자(관리자) : [물품등록, 재고확인, 물품판매]

### ⚙ 개발 환경
- 운영체제 : Windows-11
- 통합개발환경(IDE) : Intellij, STS
- JDK version : JDK 1.8
- 데이터베이스 : MySQL
- 빌드 툴 : Gradle
- 관리 툴 : GitHub
